# Katty-Farm
🐱 小猫农场 – Kitty Farm
🐱 小猫农场 – Kitty Farm
把你的键盘变成一座生机勃勃的猫咪池塘，每一次敲击都可能钓起一条小鱼，每一只小猫都在等待你的抚摸与陪伴。

🎮 游戏简介
《小猫农场》是一款桌面挂机 + 模拟养成的键盘交互游戏。它将你的物理键盘映射为一片网格池塘，每个按键都对应一格水域。你在工作、写作、编程时的每一次按键，都会被游戏视为一次“抛竿”——有概率钓到鱼，钓到的鱼可以出售换取“小鱼干”，用于收养猫咪、购买装饰。猫咪可以驻守在特定按键上，提高钓鱼效率，还能在庭院里自由活动，等待你的抚摸。

核心理念：让敲击键盘这件日常琐事，变成一次与小猫的温暖互动，让桌面成为治愈的数字港湾。

✨ 当前已实现功能
核心玩法
键盘池塘：104+ 键位完整映射，每个按键独立显示状态（🐟普通 / 🐱有猫驻守 / 💧鱼群活跃）。

钓鱼机制：普通敲击撒饵，长按或高频连击触发“鱼群爆发”，鱼获存入背包。

猫咪养成：

使用小鱼干在商店收养猫咪（最多同时拥有 4 只）。

每只猫有独立 亲密度（❤️），亲密度越高，钓鱼效率越高。

点击猫卡可 派驻 到任意键位，提升该键位的钓鱼概率。

点击 🐾 抚摸 或 拖拽鱼到猫卡 可喂食，提升亲密度并获得金币奖励。

经济系统：小鱼干（金币）和珍稀鳞片（稀有货币），钓鱼、出售、挂机均可获得。

猫咪日记：统计按键频率、最忙键位、猫咪活跃度等，生成趣味报告。

庭院视图（沉浸互动）
点击 庭院视图 按钮，从键盘池塘切换至庭院场景。

猫咪在庭院中 自由移动（随机速度、边界反弹），每只猫使用独立的 GIF 动画（box/cat*.gif）。

点击任意猫咪可 抚摸 ：

猫立即 停止运动 3 秒，形象切换为 box/catput.gif（抚摸姿态），3 秒后恢复。

头顶冒出 ❤️ 爱心动画，亲密度 +5，金币 +10。

同时触发 随机猫叫音效（从 meow*.mp3 中随机选取）。

庭院模式下 自动暂停键盘监听，避免误触；BGM 自动切换 为 bgm2.mp3（若存在）。

音效与视觉
自定义音效：支持从 sounds/ 文件夹加载 water.mp3、meow*.mp3、keypress.mp3、bgm.mp3、bgm2.mp3。

音效开关面板：可分别开关 BGM、钓鱼、猫叫、其他音效，并独立调节 BGM 音量。

钓鱼音效防抖：短时间高频敲击只播放一次，避免嘈杂。

鼠标样式：庭院中鼠标变为手套（cursor: grab），提示可互动。

界面与体验
全屏 / 窗口切换：标题栏一键切换，默认全屏启动。

浮窗模式：窗口可吸附屏幕边缘，支持多显示器。

游戏存档：所有数据保存在本地 localStorage，支持离线挂机收益（离线期间猫咪会继续捕鱼）。

可打包为独立 .exe：使用 electron-builder 生成安装程序或便携版，方便分享。

🚀 未来期望功能（社区共建）
我们欢迎开发者共同迭代，以下是我们设想的扩展方向：

🏪 商城与自定义（新）
猫咪装饰：使用小鱼干购买帽子、领结、小翅膀、墨镜等，每只猫可独立穿戴。

庭院装饰：购买秋千、猫爬架、小池塘、樱花树等，自由布置庭院场景。

池塘皮肤：更换不同水域主题（荷塘月色、深海珊瑚、樱花流水等）。

键帽皮肤：为单个或全部键位更换视觉样式（如木质、石纹、霓虹）。

🐱 猫咪系统扩展
更多猫咪品种：增加稀有猫、限定猫，丰富收集图鉴。

猫语图鉴：不同猫有独特“喵语”，通过喂食和互动解锁翻译。

流浪猫事件：随机出现流浪猫，连续敲击对应键位可收养。

🌿 生态与互动
池塘生态链：引入“大鱼吃小鱼”机制，不同鱼种互食，维持生态平衡。

好友访客猫：联网后好友猫咪可“串门”，互相赠送鱼或礼物。

季节活动：冬季池塘结冰需破冰，夏季萤火虫之夜等。

📊 数据与分享
键盘使用报告增强：生成每周/月度数据图表，支持导出和分享。

成就系统：解锁各种成就（如“连续敲击1000次”“拥有4只猫”等），获得奖励。

云存档：支持登录账号，多设备同步进度。

🎨 视觉与交互
更多 GIF 动画：为每只猫提供更多动作帧（睡觉、伸懒腰、打滚等）。

音效自定义进阶：支持用户上传自定义音效文件，实时预览。

动态背景：庭院昼夜循环、天气效果（雨、雪、阳光）。

🛠 技术栈
技术	用途
